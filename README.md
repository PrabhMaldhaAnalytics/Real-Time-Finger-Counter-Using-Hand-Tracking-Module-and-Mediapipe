📌 Real-Time Finger Counter using Hand Detector and MediaPipe

This project is a Computer Vision based application that counts the number of fingers shown in front of a webcam in real time. It uses hand detection and landmark tracking to identify finger positions and determine how many fingers are raised.

The system is built using Python, OpenCV, and MediaPipe, where MediaPipe detects the hand landmarks and OpenCV processes the video stream from the webcam. By analyzing the relative positions of finger landmarks, the program accurately counts and displays the number of fingers on the screen.

This project demonstrates the practical use of gesture recognition and human-computer interaction using real-time hand tracking technology.

🚀 Features

Real-time hand detection through webcam

Accurate finger counting using hand landmarks

Fast and lightweight computer vision system

Visual display of detected fingers on screen

🛠️ Technologies Used

Python

OpenCV

MediaPipe

Computer Vision
